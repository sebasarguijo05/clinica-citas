 Sistema de Gestión de Citas Médicas
 
Sistema web para la gestión de citas médicas orientado a clínicas pequeñas como consultorios generales y dentales.  
Desarrollado con Laravel 11. 


Descripción General

La aplicación permite administrar de forma sencilla el proceso de citas médicas, facilitando la interacción entre pacientes y administradores mediante un sistema centralizado.
Cuenta con paneles independientes para cada tipo de usuario, brindando una experiencia organizada y funcional.



Funcionalidades Principales

Paciente
- Registro e inicio de sesión
- Solicitud de citas médicas
- Visualización del estado de citas
- Cancelación de citas pendientes
- Mensajería con administración
- Visualización en calendario

Administrador
- Gestión completa de citas
- Aprobación, rechazo y reprogramación
- Administración de doctores
- Comunicación con pacientes
- Visualización general en calendario



 Tecnologías Utilizadas

- Laravel 11
- PHP
- MySQL
- Blade
- Tailwind CSS
- FullCalendar
- Integración con servicios externos (APIs)


 Estructura del Proyecto
El proyecto sigue una arquitectura basada en el patrón MVC, organizada en:

- Controladores (lógica de negocio)
- Modelos (manejo de datos)
- Vistas (interfaz de usuario)
- Rutas (definición de endpoints)


 Seguridad
El sistema implementa buenas prácticas básicas de seguridad como:

- Autenticación de usuarios
- Validación de datos
- Protección contra solicitudes maliciosas
- Manejo seguro de credenciales
